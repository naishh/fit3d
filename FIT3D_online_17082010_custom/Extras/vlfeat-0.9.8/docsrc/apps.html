Caltech-101 classification

This application uses VLFeat to train an image classifier which achieves 65% using a single feature on the Caltech-101 dataset. It uses:

  • PHOW features (dense multi-scale SIFT descriptors)
  • Elkan k-means for fast visual word dictionary construction
  • Spatial histograms as image descriptors
  • A homogeneous kernel map to transform a Chi2 support vector machine (SVM) into a linear one
  • An internal SVM (based on PEGASOS) for classification

The program is fully contained in a single MATLAB M-file, and can also be simply adapted to use your own data (change conf.calDir). The code is also bundled in VLFeat under the VLROOT/apps subdirectory.
